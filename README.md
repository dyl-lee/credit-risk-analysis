# credit-risk-analysis

## Overview
Credit card credit dataset was used in this classification problem, employing various resampling algorithms (RandomOverSampler, SMOTE, ClusterCentroids, SMOTEENN) to predict high credit risk using 2 different ensemble classifiers.
